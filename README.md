# Retail Sales Analytics Dashboard

## 📌 Project Overview

The Retail Sales Analytics Dashboard is an end-to-end data analytics project developed using Python, SQL, and Power BI. The project analyzes retail sales data to uncover business insights related to sales performance, profitability, customer segments, and regional trends.

---

## 🛠 Tools & Technologies

* Python
* Pandas
* SQL
* Power BI
* Jupyter Notebook
* Microsoft Excel

---

## 🎯 Objectives

* Analyze retail sales transactions.
* Identify sales and profit trends.
* Perform customer segmentation.
* Analyze category-wise and region-wise performance.
* Build an interactive Power BI dashboard.
